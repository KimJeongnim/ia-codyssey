# 🧠 LLM 기반 대중교통 이동 전략 설계 프로젝트

---

## 📌 프로젝트 개요

본 프로젝트는 대전정보문화산업진흥원 이동 문제를 대상으로  
LLM을 활용한 **의사결정 기반 이동 전략 생성 시스템**을 설계하고 평가한 결과물이다.

---

## 📁 파일 구조

- logs.md : 10턴 이상 대화 로그 및 환각 분석
- report.md : 모델 비교 및 평가 결과
- system_design.md : 프롬프트 및 시스템 설계

---

## 🎯 핵심 목표

- 단순 경로 안내 → 전략 기반 의사결정 시스템 전환
- 환각 최소화 구조 설계
- 비교 가능한 이동 전략 생성

---

## 📊 주요 결과 요약

- 3종 LLM 비교 수행
- 환각 발생 패턴 분석 완료
- 프롬프트 기반 구조 최적화 수행
- 문맥 유지 및 복구 구조 설계 완료

---

## 📌 결론

LLM 성능은 모델 자체보다  
👉 프롬프트 구조 + 검증 설계 + 문맥 복구 시스템에 의해 결정된다.
