## 📌 프로젝트 목적

본 프로젝트는 대전정보문화산업진흥원 이동 경로 문제를 대상으로  
LLM을 단순 질의응답 모델이 아닌 **의사결정 시스템으로 설계할 수 있는지** 검증한다.

## 🎯 핵심 결과

- LLM 성능 차이는 모델보다 “프롬프트 구조”에서 발생
- 환각은 모델 문제가 아니라 “검증 구조 부재 문제”
- 의사결정 구조 설계 시 실사용 성능 2배 이상 차이 발생
