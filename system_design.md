# 🧠 LLM 이동 전략 시스템 설계

---

## 📌 목표

대중교통 정보를 “경로”가 아닌  
👉 의사결정 가능한 전략으로 변환

---

## 📌 입력 구조

- 출발지
- 시간대
- 조건 (환승/속도/안정성)

---

## 📌 출력 구조

- 경로 2~3개
- 환승 전략
- 시간대 리스크
- 비교 구조

---

## 📌 핵심 프롬프트

당신은 이동 전략 설계 전문가이다.

- 경로 2개 이상
- 환각 금지
- 시간대 반영
- 비교 구조 필수

---

## 📌 Few-shot

출발지: 대전역

→ 1. 지하철+버스
→ 2. 버스 단일

---

## 📌 운영 전략

- 프롬프트 분리
- 상태 요약 유지
- 조건 고정

---

## 📌 문맥 복구 시스템

- 상태 요약 재주입
- 조건 잠금
- 재질문 트리거
