# 📊 LLM 모델 비교 및 업무 정의 보고서

---

## 1. 업무 과업 정의 (Criterion 1)

### 🎯 Task
대전정보문화산업진흥원까지 대중교통 이동 전략 생성

---

### 👤 Target User
- AI 네이티브 교육 수강생
- 대중교통 기반 출퇴근 사용자

---

### 📥 Input
- 출발지
- 시간대
- 이동 조건 (환승 최소 / 속도 / 안정성)

---

### 📤 Output
- 경로 2~3개
- 환승 전략
- 시간대별 리스크
- 의사결정 비교 구조

---

## 2. 입력 템플릿 (Criterion 2)

출발지: {출발지}
시간대: {시간대}
조건: {환승 최소 / 빠른 경로 / 안정성}


---

## 3. 모델 비교 (Criterion 3, 9)

| Gemini | 3.5 | 3.5 | 3.0 | 4.5 | 3.5 | 18.0 |
| Claude | 3.0 | 2.5 | 3.5 | 4.0 | 4.5 | 17.5 |

---

## 4. Few-shot 개요 (Criterion 4)

- 정상 입력 → 경로 2개 생성
- 모호 입력 → 재질문 구조 포함
- 시간대 입력 → 전략 변화

---

## 5. 모델 선정 근거 (Criterion 12)

GPT-5.4 mini 선택 이유:
- 환승 구조 설계 능력 우수
- 의사결정 구조 제공
- 교육 확장성 높음
