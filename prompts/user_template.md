# 8. AI 출력 검증 기준


AI 답변은 다음 기준으로 검토합니다.


## Pass 기준


### A. 정확한 답변

조건:

- 확인 가능한 정보 제공
- 근거 또는 확인 방법 제시


### B. 확인 필요 안내

조건:

- 정보 부족을 명확히 표시
- 추가 확인 방법 제안


---

## Fail 기준


다음 답변은 실패로 판단합니다.


- 존재하지 않는 교육 일정 생성
- 확인되지 않은 정책 작성
- 임의 비용 또는 숫자 생성
- 모호한 질문에 확인 없이 답변


---

# 9. 운영 시 활용 방법


1. 사용자는 입력 템플릿 작성

2. AI 시스템 프롬프트와 함께 전달

3. AI는 문의 분석 및 답변 생성

4. 사실 정보는 검증 절차 수행

5. 운영 담당자는 필요 시 최종 확인


---

# 10. 입력 작성 원칙


좋은 입력:

- 교육 과정명을 포함한다.
- 필요한 답변 형태를 지정한다.
- 확인이 필요한 정보를 표시한다.


나쁜 입력:

- "알려줘"
- "궁금한 게 있어"
- "교육 관련 정보 주세요"


위와 같이 정보가 부족한 경우 AI는 먼저 확인 질문을 해야 한다.
